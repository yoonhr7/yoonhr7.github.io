## iOS Safari


### 1. `<input>`  또는  `<textarea>`에 포커스될 때 **자동으로 확대(zoom)**


기본적으로 iOS는 폰트 크기가 `16px`보다 작으면 접근성을 위해 자동 확대합니다.


CSS에서 해당 인풋 요소의 `font-size`를 최소 `16px` 이상으로 설정하면 확대되지 않게 됩니다:


```css
input,
textarea {
  font-size: 16px;
}
```


**만약 시각적으로는 작은 글자를 원한다면?**


폰트 크기를 16px로 유지하되, **transform이나 scale**로 시각적으로 줄일 수 있습니다:


```css
input {
  font-size: 16px;
  transform: scale(0.875); /* 시각적으로 14px처럼 보이게 */
  transform-origin: left center;
}
```

> 단, transform: scale()을 사용할 경우 위치나 여백이 어긋날 수 있으므로 주의해서 조정해야 합니다.
