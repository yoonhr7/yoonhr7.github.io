### 프로젝트 생성

- 프로젝트 생성 및 설치

    ```plain text
    npx nuxi init my-nuxt-app // 프로젝트 생성
    
    npm i // 패키지 설치
    
    npm run serve // 개발 서버 실행
    
    npm run build // 프로젝트 빌드
    
    npm run preview // 배포할 프로젝트 확인
    ```

- 프로젝트 구조

    ```plain text
    my-nuxt-app/
    │── .nuxt/              # 빌드된 파일 (실행 중 생성)
    │── node_modules/       # npm 패키지
    │── public/             # 정적 파일 (예: 이미지, 폰트)
    │── components/         # Vue 컴포넌트 저장
    │── layouts/            # 레이아웃 파일
    │── pages/              # 페이지 파일 (자동 라우팅)
    │── plugins/            # 플러그인 (예: gsap 등록)
    │── assets/             # CSS, 이미지, 폰트 등
    │── app.vue             # 루트 Vue 파일
    │── nuxt.config.ts      # Nuxt 설정 파일
    │── package.json        # 프로젝트 설정
    ```

### 레이아웃 구성


`layout/default.vue` 생성해서 레이아웃을 구성합니다.


layout의 default.vue 가 기본 레이아웃으로 자동 설정됩니다.


```javascript
// layout/default.vue
<template>
    <div>
        <header class="bg-blue-500 text-white p-4">
            <h1>내 사이트</h1>
        </header>
        <main>
            <slot />
        </main>
        <footer class="bg-gray-800 text-white p-4 text-center">ⓒ 2025 My Site</footer>
    </div>
</template>
```


✔️ `<slot />` 영역에 해당하는 페이지 내용이 들어갑니다.


Nuxt3에서는 `app.vue`가 최상위 파일이기 때문에 `app.vue`를 수정해야합니다.


```javascript
// app.vue
<template>
    <NuxtLayout>
        <NuxtPage />
    </NuxtLayout>
</template>
```

### gsap 사용

- **gsap 설치**

    ```javascript
    npm i gsap
    ```

- **`plugins/gsap.ts`** **에 플러그인으로 저장하기**

    ```typescript
    // plugins/gsap.ts
    import { gsap } from "gsap";
    import { ScrollTrigger } from "gsap/ScrollTrigger";
    
    export default defineNuxtPlugin((nuxtApp) => {
        gsap.registerPlugin(ScrollTrigger);
    
        return {
            provide: {
                gsap,
            },
        };
    });
    ```

### SEO 설정

- **`app.vue`****에서 전역** **`title`** **및** **`meta`** **설정** (가장 간단한 방법)

    ```typescript
    <script setup>
    import { useHead } from 'unhead';
    
    useHead({
      title: '기본 사이트 제목',
      meta: [
        { name: 'description', content: '이 웹사이트는 Nuxt 3으로 제작되었습니다.' },
        { property: 'og:title', content: '기본 사이트 제목' },
        { property: 'og:description', content: 'Nuxt 3에서 메타 태그 설정하기' }
      ]
    });
    </script>
    
    <template>
      <NuxtPage />
    </template>
    ```

- **`plugins`** **폴더에서 커스텀** **`useSeo`** **훅 만들기**

    반복적으로 사용되는 `title`, `meta` 등을 **재사용 가능하게** 만들고 싶다면, `plugins/useSeo.ts` 파일을 만들어서 사용하면 좋습니다.

    - **플러그인 설정하기 (****`plugins/useSeo.tx`****)**

        ```typescript
        // plugins/useSeo.ts
        export const useSeo = (title: string, description: string) => {
          useHead({
            title,
            meta: [
              { name: 'description', content: description },
              { property: 'og:title', content: title },
              { property: 'og:description', content: description }
            ]
          });
        };
        ```

    - **페이지에서 사용하기 (****`pages/index.vue`****)**

        ```typescript
        // pages/index.vue
        <script setup>
        import { useSeo } from '~/plugins/useSeo';
        
        useSeo('홈페이지 | 내 사이트', '이것은 내 Nuxt 홈페이지입니다.');
        </script>
        
        <template>
          <div>
            <h1>홈페이지</h1>
          </div>
        </template>
        ```


    → **`useSeo('페이지 제목', '설명')`****으로 간단하게 설정 가능**


    → 여러 페이지에서 **재사용 가능**

- **`nuxt.config.ts`****에서** **`app.head`** **설정 (전역 기본값)**

    `nuxt.config.ts`에서 `app.head` 설정을 하면 **앱 전체의 기본** **`title`****과** **`meta`** **값을 설정할 수 있어.**


    ```typescript
    export default defineNuxtConfig({
      app: {
        head: {
          title: '기본 사이트 제목',
          meta: [
            { name: 'description', content: '이 웹사이트는 Nuxt 3으로 제작되었습니다.' }
          ]
        }
      }
    });
    ```


    →  **모든 페이지에 기본** **`title`** **&** **`meta`** **적용**


    →  **각 페이지에서 덮어쓰기 가능**


### **최종 정리**


| 방법                      | 사용 위치  | 특징                             |
| ----------------------- | ------ | ------------------------------ |
| **`app.vue`**           | 전역     | 모든 페이지에서 기본 `title`, `meta` 적용 |
| **`plugins/useSeo.ts`** | 재사용 가능 | 여러 페이지에서 공통 SEO 설정을 쉽게 적용 가능   |
| **`nuxt.config.ts`**    | 전역 기본값 | 모든 페이지에 공통 `title`, `meta` 제공  |

- **빠르게 적용**하려면 → `app.vue`에서 `useHead` 사용
- **SEO 설정을 재사용**하려면 → `plugins/useSeo.ts` 만들기
- **전체 사이트의 기본 설정**은 → `nuxt.config.ts`에서 `app.head` 설정
