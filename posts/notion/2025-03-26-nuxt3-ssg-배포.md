### 1. `nuxt.config.ts`에 SSG로 빌드할 수 있도록 설정


```typescript
// nuxt.config.ts
export default defineNuxtConfig({
  ssr: false, // 클라이언트 사이드 렌더링
  nitro: {
    preset: "static",
  }
});
```


### 2. build


```shell
npm run build
```
